# Contributing to AEGIS

Thank you for your interest in contributing to AEGIS, a limitless AI expert for IT and Cybersecurity. I welcome all positive and constructive contributions.

## How to Contribute

- **Report Bugs:** If you find a bug, please open an issue with a clear description.
- **Suggest Features:** If you have an idea for a new feature, please open an issue to discuss it.
- **Submit Code:** If you want to write code, please fork the repository and submit a pull request with a clear explanation of your changes.

## Guidelines

- Please be respectful and constructive in all communications.
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md).
- For security vulnerabilities, please follow the [Security Policy](SECURITY.md).

Thank you for helping make AEGIS even better.
